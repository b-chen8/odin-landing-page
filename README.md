# Odin Landing Page

A static landing page built as part of [The Odin Project](https://www.theodinproject.com/) curriculum. The goal was to recreate a provided design mockup using only HTML and CSS.

## Preview

![Landing page design reference](01.png)

## Features

- Responsive-style layout built with Flexbox
- Four distinct sections: hero, information, quote, and call-to-action
- Dark/light section contrast using a consistent color palette
- Google Fonts (Roboto)

## Tech Stack

- HTML5
- CSS3 (Flexbox)

## Design Specs

The page follows the design spec from `02.png`:

| Element | Value |
|---|---|
| Font | Roboto |
| Dark background | `#1F2937` |
| Hero main text | 48px, `#F9FAF8`, weight 900 |
| Hero secondary text | 18px, `#E5E7EB` |
| Button / CTA background | `#3882F6` |
| Quote section background | `#E5E7EB` |
| Quote text | 36px, `#1F2937`, weight 300 |

## Getting Started

No build steps needed. Open `index.html` directly in a browser.
